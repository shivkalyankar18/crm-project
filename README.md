# CRM Project (Customer Relationship Management)

## 📌 Project Description
This is a Customer Relationship Management (CRM) system developed using Django.  
It helps manage customer records, user authentication, and business data efficiently.

---

## 🚀 Features
- User Registration & Login
- Add New Customer Records
- View Customer Details
- Update Customer Information
- Delete Records
- Dashboard Interface

---

## 🛠️ Technologies Used
- Python
- Django
- SQLite
- HTML, CSS, JavaScript

---

## 📂 Project Structure
crm/
│── crm/
│── crmapp/
│── static/
│── templates/
│── manage.py

---

## ▶️ How to Run the Project

1. Install Django:

pip install django


2. Run migrations:

python manage.py migrate


3. Start server:

python manage.py runserver


4. Open browser:

http://127.0.0.1:8000/
